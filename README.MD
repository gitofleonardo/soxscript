# SoxScript

> ![](raw/soyorin-s.jpg) 长期 sox 导致的

# 这是什么

一门简单的脚本语言

# 特性

- [x] 图灵完备
- [x] 支持基本类型
- [x] 支持基本容器（数组、映射）
- [x] 模板字符串
